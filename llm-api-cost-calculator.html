How LLM API pricing works

Most providers bill per token (~¾ of a word), with separate input and output rates. Monthly cost = requests × (avg input tokens × input price + avg output tokens × output price). Reasoning and "Pro" models cost far more per token, while small models (GPT-4o-mini, Gemini Flash, DeepSeek V4) are a fraction of the price.
